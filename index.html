About

A comprehensive Python-based pricing engine for weather derivatives with focus on rainfall-based contracts for the construction industry. Developed for Cliff Horizon Pte. Ltd., a Singapore-based weather intelligence services company.

Key Features

Distribution Fitting

Automatic selection between Poisson, Negative Binomial, and Empirical distributions with AIC-based optimization

Monte Carlo Simulation

10,000+ scenario analysis with bootstrap, parametric, and hybrid methods for robust pricing

Risk Metrics

Comprehensive VaR, CVaR, trigger frequency analysis, and capital requirement calculations

Pricing Engine

Complete premium calculation with risk loadings, capital charges, and profitability analysis

Contract Validation

Input validation and contract specification management with type safety

Professional Output

Formatted summaries, detailed breakdowns, and export-ready results

Basic Usage

from simulator import WeatherDerivativeSimulator from config import ContractSpecification import numpy as np # Create simulator sim = WeatherDerivativeSimulator() # Define contract contract = ContractSpecification( location_name="Bankot, India", observation_month=10, rainfall_threshold_mm=25.0, strike_rainy_days=11, payout_rate_per_day=14000, maximum_payout_days=7 ) # Historical data (10 years) historical_data = np.array([7, 10, 9, 12, 8, 14, 9, 11, 8, 10]) # Price the contract results = sim.price_contract(contract, historical_data) print(sim.generate_summary())
